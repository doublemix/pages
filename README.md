## About

This repository contains a portfolio of personal projects built with modern web technologies. The homepage provides easy navigation to all included applications.

### Featured Projects

- **Yard Sale Tracker App** - An application for tracking yard sale items and transactions
- **Restaurant Speed Run** - A fun restaurant-themed application
- **Ranking App** - An application for ranking and comparing items

## Development

The projects in this repository are built with various technologies including:
- HTML/CSS with Tailwind CSS
- JavaScript
- Vite (for the Yard Sale Tracker)

### Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the `master` branch.
